# Osemsmerovka

### Projekt 1 | Predmet:  Základy procedurálneho programovania 2

## Funkcionalita

- osemsmerovka bude zadaná v súbore osemsmerovka.txt.
 - obsahuje v prvom riadku 2 celé čísla m n predstavujúce rozmer obdĺžnikovej štvorsmerovky/osemsmerovky.
 
 ### Program obsahuje
 -	 Dvojrozmerné dynamické obdĺžnikové pole s rozmermi mxn, kam načíta osemsmerovku.
 -  Index - 26 prvkové pole (každý prvok pre jedno písmeno) ukazovateľov na polia obsahujúce indexy pre výskyty daného písmena v osemsmerovke.
